# Visualisierung langfristiger HRV-Aenderungen

## Verwendung
Um die Graphen zu erstellen, führen Sie bitte main.m unter src/ aus. 
Die Graphen werden unter results/ gespeichert.

## Addons
- Signal Processing Toolbox (MathWorks)
- Statistics and Machine Learning (MathWorks)

## Structure
- results/ - Stores the generated graphs
- data/ - Stores the EDF files
- src/ - stores the source code
